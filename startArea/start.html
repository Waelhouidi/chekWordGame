Rules of the game

Deroulement de jeu :

  • Le joueur dispose de 6 essais pour deviner le mot caché
  • À chaque essai, le joueur saisit une lettre dans la grille pour former un mot.
  • Après chaque essai, le jeu indique les lettres correctes placées à la bonne position, les lettres correctes placées à la mauvaise position et les lettres incorrectes.

Fin de partie :

  • La partie se termine lorsque le joueur devine correctement le mot ou lorsqu'il a utilisé ses 6 essais sans succès
  • En cas de réussite, le jeu affiche un message de victoire.
  • En cas d'échec, le jeu affiche le mot correct.
